# 🚀 SpeedFlow

Sistema de organização de rotina acadêmica desenvolvido para auxiliar estudantes universitários no planejamento, acompanhamento e gerenciamento de tarefas.

## 📚 Sobre o Projeto

O SpeedFlow foi criado com o objetivo de ajudar estudantes a organizarem suas atividades acadêmicas, evitando atrasos, reduzindo a sobrecarga de estudos e facilitando o acompanhamento do progresso semanal.

O projeto foi desenvolvido utilizando React, CSS e LocalStorage, sem utilização de back-end.

---

## ✨ Funcionalidades

### 📋 Gerenciamento de Tarefas

Permite:

- Criar tarefas
- Excluir tarefas
- Concluir tarefas
- Definir prioridade
- Definir dia da semana
- Definir tempo estimado

---

### 📊 Dashboard Inteligente

Exibe:

- Total de tarefas
- Tarefas concluídas
- Tarefas pendentes
- Tarefas de alta prioridade
- Progresso semanal

---

### ⚡ Indicador de Sobrecarga

Calcula automaticamente a carga semanal baseada no tempo planejado e classifica o usuário em:

- Leve
- Moderada
- Alta

---

### 📅 Plano Semanal

Organiza as tarefas automaticamente por dia da semana, facilitando o planejamento dos estudos.

---

### 🏆 Priorização Automática

As tarefas são ordenadas automaticamente por prioridade:

1. Alta
2. Média
3. Baixa

---

### 💡 Resumo Inteligente

Exibe:

- Próxima tarefa prioritária
- Horas planejadas
- Quantidade de tarefas pendentes
- Mensagem motivacional

---

### 🎉 Sistema de Notificações

Notificações visuais personalizadas para:

- Criação de tarefas
- Conclusão de tarefas
- Exclusão de tarefas
- Validação de campos

---

### 🤖 Planejamento Automático

Sugere uma distribuição equilibrada para tarefas extensas, auxiliando o usuário na organização da rotina.

---

### 📱 Responsividade

Interface adaptada para:

- Desktop
- Tablets
- Smartphones

## ▶️ Como Executar Localmente
### 1. Clonar o repositório
```bash
git clone URL_DO_REPOSITORIO
```
### 2. Entrar na pasta do projeto
```bash
cd Projeto-Front-End
```
### 3. Instalar dependências
```bash
npm install
```
### 4. Executar o projeto
```bash
npm run dev
```
### 5. Abrir no navegador
```bash
http://localhost:5173
```
